**Welcome to the Framework Evolution Lab** 🧬

Step into the neon-lit corridors of frontend history! This lab takes you on an epic journey through time — from the golden age of AngularJS 1.6 (circa 2010) to the cutting-edge frontier of **React 18** or **Angular 17** with TypeScript.

## 💾 LEGACY STACK (2010 Era)

### The Old Guard 🗡️

| Technology | Version | Status | Reason for Replacement |
|------------|---------|--------|------------------------|
| 🅰️ **AngularJS** | 1.6.x | ❌ **EOL (2022)** | No security patches, poor performance, no TypeScript |
| 📦 **Bower** | 1.8.x | ❌ **Deprecated** | Unmaintained since 2017, replaced by npm |
| ⚙️ **Grunt** | 1.x | ⚠️ **Legacy** | Slow builds, complex config, replaced by Vite/CLI |
| 💲 **jQuery** | 2.x | ⚠️ **Not needed** | DOM manipulation done by React/Angular |
| 🔧 **Restangular** | 1.6.x | ⚠️ **Abandoned** | Extra abstraction layer over $http |
| 📅 **Moment.js** | 2.x | ⚠️ **Deprecated** | Huge bundle size (67KB), use date-fns (12KB) |
| 🔨 **Lodash** | 4.x | ⚠️ **Partially needed** | Many functions available in modern ES6+ |
| ☕ **Karma + Jasmine** | - | ⚠️ **Outdated** | Replaced by Vitest (React) or modern Jasmine (Angular) |

### Classic Anti-Patterns Present 🐛

- ❌ `$scope`-based data binding (not `controllerAs` syntax)
- ❌ `$rootScope` events for cross-component communication (`$broadcast`/`$on`)
- ❌ jQuery plugins wrapped in AngularJS directives (DOM manipulation conflicts)
- ❌ `$watch` and `$watchCollection` for manual change detection
- ❌ Two-way data binding causing performance issues on large lists
- ❌ No TypeScript — dynamic typing leading to runtime errors
- ❌ Mixed jQuery and AngularJS DOM manipulation
- ❌ Global namespace pollution

#### **🧩 STAGE 4: MIGRATE COMPONENTS** (2-3 hours)

**Objective:** Convert AngularJS controllers + templates to React/Angular components.

**Migration Pattern (React):**

**Before (AngularJS Controller):**
```javascript
// flight-search.controller.js
angular.module('app').controller('FlightSearchController', 
  function($scope, FlightService) {
    $scope.searchParams = { origin: '', destination: '' };
    $scope.flights = [];
    
    $scope.searchFlights = function() {
      FlightService.search($scope.searchParams).then(function(data) {
        $scope.flights = data;
      });
    };
  }
);
```

**After (React Component):**
```typescript
// FlightSearch.tsx
import { useState } from 'react';
import { useQuery } from '@tanstack/react-query';
import { searchFlights } from '@/api/flightService';

export const FlightSearch = () => {
  const [searchParams, setSearchParams] = useState({ origin: '', destination: '' });
  
  const { data: flights, refetch } = useQuery({
    queryKey: ['flights', searchParams],
    queryFn: () => searchFlights(searchParams),
    enabled: false,
  });

  return (
    <div>
      {/* Component JSX */}
    </div>
  );
};
```

**Migration Pattern (Angular):**

**After (Angular Standalone Component):**
```typescript
// flight-search.component.ts
import { Component, signal } from '@angular/core';
import { FlightService } from '@/services/flight.service';

@Component({
  selector: 'app-flight-search',
  standalone: true,
  templateUrl: './flight-search.component.html',
})
export class FlightSearchComponent {
  searchParams = signal({ origin: '', destination: '' });
  flights = signal([]);

  constructor(private flightService: FlightService) {}

  searchFlights() {
    this.flightService.search(this.searchParams()).subscribe(
      data => this.flights.set(data)
    );
  }
}
```

#### **🏪 STAGE 5: MIGRATE STATE MANAGEMENT** (1-2 hours)

**Objective:** Replace `$scope` and `$rootScope` with modern state management.

**Before (AngularJS):**
```javascript
// Using $rootScope for global auth state
$rootScope.$on('user:loggedIn', function(event, user) {
  $scope.currentUser = user;
});

$rootScope.$broadcast('user:loggedIn', userData);
```

**After (React + Zustand):**
```typescript
// store/authStore.ts
import { create } from 'zustand';

export const useAuthStore = create((set) => ({
  currentUser: null,
  login: (user) => set({ currentUser: user }),
  logout: () => set({ currentUser: null }),
}));

// Use in components
const { currentUser, login } = useAuthStore();
```

**After (Angular + NgRx):**
```typescript
// store/actions/auth.actions.ts
import { createAction, props } from '@ngrx/store';

export const loginSuccess = createAction(
  '[Auth] Login Success',
  props<{ user: User }>()
);

// Use in components
this.store.dispatch(loginSuccess({ user }));
this.currentUser$ = this.store.select(selectCurrentUser);
```

#### **🔌 STAGE 7: REPLACE DEPENDENCIES** (1 hour)

**Objective:** Remove legacy libraries and use modern alternatives.

**Replacements:**
- ❌ jQuery → ✅ Native React/Angular (no DOM manipulation needed!)
- ❌ Moment.js → ✅ date-fns
- ❌ Lodash → ✅ Native ES6+ (map, filter, reduce, etc.)
- ❌ Restangular → ✅ TanStack Query + Axios (React) or HttpClient (Angular)

**Example (date-fns):**
```typescript
// Before (Moment.js) - 67KB bundle
import moment from 'moment';
const formattedDate = moment(date).format('YYYY-MM-DD');

// After (date-fns) - 12KB bundle
import { format } from 'date-fns';
const formattedDate = format(date, 'yyyy-MM-dd');
```

```bash
# Solution: Use legacy peer deps
npm install -g bower --legacy-peer-deps
```

**Problem:** Grunt build is slow


**Problem:** React app won't compile
```bash
# Solution: Clear cache and reinstall
rm -rf node_modules package-lock.json
npm install
npm run dev
```

```bash
# Solution: Install Angular 17 globally
```
